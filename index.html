Airflow DAGs

File: dags/http_dag.py · dags/db_dag.py · dags/file_dag.py

Input: Environment variables (API_URL, DB_PATH, etc.)

Output: Task execution logs via REST API

Three injectable Apache Airflow DAGs performing HTTP ETL, SQLite DB reads, and file processing. All configuration is read from os.environ so failures can be injected without touching DAG code.

// 03 FAILURE TAXONOMY

5 failure classes, fully labeled

⏱

timeout

http_error

DAG: http_dag

HTTPError: 404 Client Error: Not Found for url: https://httpbin.org/status/404

Repair: Fix API_URL
📂

missing_file

DAG: file_dag

FileNotFoundError: [Errno 2] No such file or directory: '/nonexistent/path/data.csv'

Repair: replace_path
🗃

missing_column

DAG: db_dag

KeyError: 'nonexistent_column' in processed results.

Repair: set_env column override
🗄

missing_db

DAG: db_dag

OperationalError: unable to open database file: /tmp/nonexistent.db

Repair: fix DB_PATH
